# HOW TO PLAY
- Run Echoes of The Deep.exe to play
- W, A, S, D to move and attack
- The goal is to get a high a score/ level as possible.
- Based on how you performed last game, the difficulty will adjust.

# NOTICE
- This a test build so many bugs, mis-balancing, incomplete or missing features are to be expected.
